Prueba de Vacío en Soldaduras

La prueba de vacío se utiliza generalmente para detectar fugas en las juntas de soldadura. Se realiza utilizando una caja de prueba con una ventana transparente en la parte superior, que proporciona una visibilidad adecuada para ver el área bajo examen. Se proporcionan conexiones, válvulas, iluminación y medidores, según se requiera. Se utiliza una solución de película de jabón o una solución de detección de fugas comercial, aplicable a las condiciones.
